Programming Bootcamp
This was another one of our university course where we learned how to program based on JAVA while using the "Processing" text editor. Every lecture, we were introduced to more and more functionality of programming. Then we were given assignments to create program which fit the general instructions.
Assignment 1: Dynamic Drawing
This image was came from a code running which the purpose was to create an artwork, but every time it ran, it produced a different form. The product had to have a pattern but different.
Assignment 2: Game
This is a simple game created by coding. The purpose of the game is to get the ball from the left to the right. If the ball touches the boxes, the you lose the game. Every time the game runs, the boxes are in different order.
